# Система академического мониторинга студентов

Система для учета успеваемости, оценок и посещаемости студентов с различными уровнями доступа для студентов и старост.

## Функциональность

### Для студентов:
- Просмотр своих оценок по предметам
- Просмотр посещаемости занятий
- Расчет среднего балла по предметам
- Статистика успеваемости
- Формирование персонального отчета в JSON

### Для старост:
- Просмотр успеваемости всей группы
- Добавление оценок студентам
- Отметка посещаемости занятий
- Статистика группы (средние баллы, успеваемость)
- Формирование отчета группы в JSON

## Технологии
- Python 3.x

- SQLite - база данных

Встроенные библиотеки Python - нет внешних зависимостей

## Установка и запуск
Создайте папку для проекта

Сохраните все файлы в этой папке

Запустите программу:

```bash
python main.py
```
